# Compilers 2021/2022

### Projekt 1
Program sprawdza, czy podany ciąg znaków pasuje do wyrażenia regularnego: (aa|bb)?bb(aa|b)*

### Uruchomienie
1. Instalacja Dart SDK w wersji 2.17.1 (stable)
2. `git clone https://github.com/elefantaka/compilers_regexp.git`
3. `cd regexp`


### Testy
`run test -r json`
